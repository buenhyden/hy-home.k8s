## Overview

이 문서는 효율적인 AI Agent 활용을 위한 표준 워크플로우를 정의한다.

## Core Workflows

1. **Feature Development**:
   - PRD 검토 → Spec 작성 → Plan 수립 → Task 수행 → 검증
2. **Issue Resolution**:
   - 상황 파악 → 원인 분석(RCA) → 수정 계획 → 적용 → 검증
3. **Documentation**:
   - 템플릿 선택 → 초안 작성 → 거버넌스 준수 검토 → 최종 승인

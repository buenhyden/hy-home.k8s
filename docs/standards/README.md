# Project Standards

This directory defines the standards and conventions that ensure consistency across the `hy-home.k8s` platform.

## Available Standards

### 1. [Naming Conventions](./naming-conventions.md)

Rules for naming Kubernetes resources, files, directories, and Git branches.

### 2. [GitOps Standard](./gitops-standard.md)

The formal process for submitting changes and how ArgoCD syncs them to the cluster.

### 3. [Security Policy](./security-policy.md)

Governance rules enforced by Kyverno, Network Policies, and RBAC guidelines.

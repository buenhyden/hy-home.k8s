# Operational Guides

This directory contains practical, step-by-step guides for managing the `hy-home.k8s` platform.

## Available Guides

### 1. [Cluster Bootstrap](./cluster-bootstrap.md)

Detailed instructions on how to initialize the local Kubernetes environment (Kind), install ArgoCD, and sync the root applications.

### 2. [Adding Applications](./adding-applications.md)

How to use the built-in templates to deploy new backend or frontend services using GitOps.

### 3. [Secret Management](./secret-management.md)

Guidelines and commands for encrypting secrets with Sealed Secrets to safely store them in Git.

### 4. [Development Workflow](./development-workflow.md)

Best practices for setting up your environment, making changes, and contributing to the project.

### 5. [Disaster Recovery](./disaster-recovery.md)

Procedures for recovering from cluster failures and restoring state.

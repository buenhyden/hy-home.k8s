# Agent Memory System

Guidelines for recording and retrieving agent-learned outcomes.

## 1. Structure

- `/README.md`: Index and categorized list of memories.
- `/template.md`: Standard format for new memory entries.
- `/<YYYY-MM-DD>_<slug>.md`: Individual memory files.

## 2. Usage Policy

- **Record**: Post-mortem or after resolving a complex, non-obvious issue.
- **Search**: Step 5 of the JIT Protocol (Context-Aware Loading).
- **Format**: Follow `template.md` (Problem, Context, Solution, Prevention).

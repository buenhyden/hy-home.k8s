# Documentation Protocol (March 2026)

This protocol defines how agent governance references authored docs.

## Core Requirements

- Governance policy belongs in `docs/00.agent-governance/`.
- Product and delivery truth remains in `docs/01~99`.
- Governance files must reference authored docs, not rewrite them.

## Traceability Rules

- Every governance change should keep clear links to stage folders (`01` to `11`, `90`, `99`).
- Persona and scope instructions must state which stage folders are authoritative.

## README Rules

- `README.md` and `docs/README.md` must remain Korean.
- Governance folder docs must remain English.
- Language policy must be documented in both root README files.

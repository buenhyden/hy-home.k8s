## [0.1.0] - 2026-01-13

### Added

- Initial project structure for GitOps-based Kubernetes infrastructure platform.
- Cluster setup scripts in `bootstrap/`.
- ArgoCD integration and root applications configuration.
- Platform services: Istio, Prometheus Stack, Loki, Tempo, Alloy, Kyverno, Cert-Manager, Sealed Secrets.
- External services: PostgreSQL (Patroni), Redis Cluster.
- Automated updates with Dependabot.
- Issue and Pull Request templates for GitOps workflow.
